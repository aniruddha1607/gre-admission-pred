Frequently asked Questions

What does this web application do?

We basically have a dataset with the following parameters like GRE score, TOEFL score etc and gives the output that is chance of admit in the range of 0 to 1. This dataset is then trained using our ML model which is currently using linear regression to predict the target column(Chance of Admit) and provide you with an idea about the chances in your favourite uni!
